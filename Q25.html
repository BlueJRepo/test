A    Each Batch Job uses a SINGLE THREAD to process a configured block size of record.
            Each Batch Step instance receives a BLOCK of records as payload, and BLOCK of records are processed in ORDER.

B    Each Batch Job uses a SINGLE THREAD for all Batch Steps.
            Each Batch Step instance receives ONE record at a time as the payload, and RECORDS are processed IN ORDER, first through Batch_Step_1 and then through Batch_Step_2.

C    Each Batch Job uses SEVERAL THREADS for the Batch Steps.
            Each Batch Step instance receives ONE record at a time as the payload, and BATCH STEP INSTANCES executes IN PARALLEL process records and Batch Steps in ANY order as fast as possible.

D    Each Batch Job uses SEVERAL THREADS for the Batch Steps.
            Each BATCH Step instance receives ONE record at a time as payload, and process IN PARALLEL within and between the two Batch Steps.
