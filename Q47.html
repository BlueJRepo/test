A    Order messages are sent to JMS queue. Order fulfilled messages are sent to a JMS topic.
            Both microservices interact with same JMS provider(message broker).

B    Order messages are sent directly to the Fulfilment microservice. OrderFulfilled messages are sent directly to the Order microservice.
            The order AMQP-compatible message broker, so that message brokers can be chosen and scaled to best support the load of the each microservice.

C    Order messages are sent to JMS queue. Order fulfilled messages are sent to JMS topic.
            The order microservice interacts with one JMS provider (message broker) and the fulfillment microservice with different JMS provider,
            so both message brokers can be chosen and scaled to best support the load of the each microservice

D    Order message are sent to an Anpoint MQ exchange. Order fulfilled message are spent to an Anypoint MQ queue.
            Both microservice interact with Anypoint MQ as the message broker, which must therefore scale to support the load of both microserivce.
